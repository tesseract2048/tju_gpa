tju_gpa
=======

Undergraduate GPA Calculator for (probably more than) TJU


## usage

python gpa.py -s ***\<school\>*** -c ***\<scale\>*** -u ***\<username\>*** -p ***\<password\>*** [-t ***\<term\>***]

where term is optional. see python gpa.py -h for help.


## supported schools

* tju


## supported scales

* tju
